# Scrapy
## Web Crawler using `Scrapy` package on python 3.x

### How to install package ?
`python3 -m pip install Scrapy`

### How to create project from `Scrapy` ?
- Move to folder

`cd FolderName`
- Create `Scrapy` project

`scrapy startproject yourprojectname`
